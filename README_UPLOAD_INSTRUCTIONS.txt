Daily Dose Website — Devotions 1–39 Upload Package

Created from: DAILY DOSE 1.docx
Included devotions: 1–39

Upload instructions:
1. Go to Cloudflare → Workers & Pages → dailydosedevotions → Create deployment.
2. Keep Production selected.
3. Upload the full contents of this folder or upload the ZIP file directly if Cloudflare accepts ZIP upload.
4. Click Save and deploy.
5. After success, visit: https://dailydosedevotions.ie/devotions.html

Files included:
- index.html
- devotions.html
- styles.css
- script.js
- /devotions/daily-dose-1.html through /devotions/daily-dose-39.html

Note: Devotions 1–39 are archive/live pages. Future scheduled devotions should be handled in a separate scheduling workflow.
